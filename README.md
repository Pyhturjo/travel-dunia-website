# Travel Dunia 🌍

A responsive, static travel website built with **HTML, CSS, and JavaScript**.  
This project showcases destinations, packages, and basic UI interactions for a travel brand concept called **Travel Dunia**.

🔗 Live : traveldunia.com.bd

---

## ✨ Features

- Responsive layout (mobile, tablet, desktop)
- Hero section with CTA
- Destination / package cards
- Smooth scrolling navigation
- Simple interactive elements
- Fast load, no frameworks

## 📁 Project Structure

```
travel-dunia-website/
├── index.html
├── style.css
├── main.js
├── assets/
│   ├── images/
│   └── icons/
├── README.md

```

## 🛠️ Tech Stack

* **HTML** for structure
* **CSS** for layout & responsiveness

## 🚀 Run Locally

```bash
git clone https://github.com/Pyhturjo/travel-dunia-website.git
cd travel-dunia-website
```

## 🧩 Customization Guide

* **Text & sections**: Edit `index.html`
* **Colors & layout**: Edit `style.css`
* **Interactions**: Edit `main.js`
* **Images**: Replace files in `assets/images/`

Want a different hero image or package list? Swap the image paths and card content in HTML. That’s it.

---

## 📦 Deployment (GitHub Pages)

This site is deployed using **GitHub Pages**.

If you want to redeploy:

1. Push changes to `main` branch
2. Go to **Repo → Settings → Pages**
3. Set source to `main` / root
4. Save and wait ~1 minute

Your site updates automatically.
